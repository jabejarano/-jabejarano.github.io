Descripción del Proyecto

Este proyecto se enfoca en la construccion de un conjunto de componentes para crear menús interactivos en realidad virtual, utilizando el seguimiento de manos como metodo exclusivo de interaccion. La importancia radica en que permite la interacción sin controladores, haciendo uso unicamente de las manos.
